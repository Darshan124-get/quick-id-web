# Screenshot Setup Guide

## Adding Your 12 Application Screenshots

To add your 12 application screenshots, simply place them in the `images/` folder with the following naming convention:

### Required Files:
1. `screenshot1.jpg` - Photo Capture Interface
2. `screenshot2.jpg` - Student Management
3. `screenshot3.jpg` - ID Card Preview
4. `screenshot4.jpg` - Student Search
5. `screenshot5.jpg` - Bulk Upload
6. `screenshot6.jpg` - Photo Editor
7. `screenshot7.jpg` - Class Management
8. `screenshot8.jpg` - Reports Dashboard
9. `screenshot9.jpg` - Settings Panel
10. `screenshot10.jpg` - Export Options
11. `screenshot11.jpg` - Mobile App
12. `screenshot12.jpg` - Admin Dashboard

### Image Specifications:
- **Format**: JPG, PNG, or WebP
- **Recommended Size**: 300x500px (portrait) or 500x300px (landscape)
- **File Size**: Keep under 200KB per image for fast loading
- **Aspect Ratio**: 3:5 or 5:3 works best

### Tips for Great Screenshots:
- Use high-resolution screenshots
- Ensure good contrast and readability
- Show the most important features clearly
- Consider adding subtle annotations or highlights
- Make sure text is readable in the thumbnails
- Use consistent styling across all screenshots

### Testing:
After adding your screenshots:
1. Open `index.html` in your browser
2. Navigate to the "Screenshots" section
3. Verify all 12 images load correctly
4. Test on different screen sizes to ensure responsiveness
5. Check that hover effects work properly

The website will automatically handle the layout and responsive design for all 12 screenshots!
